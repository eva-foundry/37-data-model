## READ

**What We've Learned**:

1. **GitHub Models Free Tier Suffices**: gpt-4o-mini (free) + gpt-4o (free-limited 5%) = $0.007/sprint. Claude NOT available in free tier.

2. **Correlation ID System Works**: Format `SPRINT-{id}-{timestamp}-{uuid[:8]}` never collides. End-to-end traceability via log prefix `[TRACE:...]`.

3. **LM Calls Must Be Tracked At Injection Point**: Not post-hoc. LMTracer integrates at call site (bug_fix_agent.py), captures tokens + cost immediately.

4. **Cost Calculation Needs Precision**: GitHub Models formula: `(tokens_in × input_mult + tokens_out × output_mult) × $0.00001`. Use 8-decimal precision, not rounding.

5. **Three-Phase Bug Fix Has Different Costs**: Phase A (RCA ~3KB prompt) < Phase B (Fix ~5KB) < Phase C (Test ~2KB). Separate traces per phase enable optimization.

6. **Data Model as Single Source of Truth Works**: 28 layers (27 existing + traces) = production infra ready. Cosmos persistence (ACA) + MemoryStore (local) both work.

7. **Evidence Schema Evolution Possible**: Added timeline (6 fields), lm_interaction, WBS hierarchy without breaking existing records. JSON Schema validation prevents drift.
